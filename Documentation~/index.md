# Cloud Code SDK documentation
Consult the full Cloud Code manual at [https://docs.unity.com/cloud-code/](https://docs.unity.com/cloud-code/).

# Cloud Code authoring
Cloud Code assets can be created, modified, and uploaded from the Unity Editor. For more information, refer to the following pages in the Cloud Code manual:

- Cloud Code scripts: [Modify script parameters](https://docs.unity.com/en-us/cloud-code/scripts/how-to-guides/write-scripts/unity-editor)
    - JavaScript project: [Cloud Code JavaScript projects in the Unity Editor](https://docs.unity.com/en-us/cloud-code/scripts/how-to-guides/write-scripts/unity-editor#cloud-code-javascript-projects-in-the-unity-editor)
- Cloud Code modules: [Using Unity Editor](https://docs.unity.com/en-us/cloud-code/modules/how-to-guides/write-modules/unity-editor)
